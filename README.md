## Description
The Electricity Billing System is a Java-based application designed to calculate electricity charges based on the number of units consumed by a customer. The system accepts customer details and unit consumption as input, applies predefined tariff rates, and generates the final electricity bill. This project demonstrates the use of conditional statements, user input handling, and arithmetic calculations in Java.

## Features
- Customer Information Management
- Electricity Bill Calculation
- Unit-Based Tariff System
- User-Friendly Console Interface
- Accurate Billing Output

## Technologies Used

- Scanner Class
- Conditional Statements
- Arithmetic Operations
